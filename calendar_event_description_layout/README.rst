The default calendar event form provides limited space for the
Description field, which can be inadequate when detailed information is
required.

This constraint negatively impacts usability in scenarios involving long
or complex descriptions.

To address this issue, this addon adjusts the layout of the calendar
event form by placing the description field in its own page.
